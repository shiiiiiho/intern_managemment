# Task Complete コマンド

タスク完了時の処理を自動化するコマンド。

## 実行フロー

1. **事前チェック**
   - 全変更がコミット済み
   - 全テストがパス

2. **変更内容の分析**
   - mainからの差分を確認
   - コミット履歴を確認

3. **リモートにプッシュ**

   ```bash
   git push -u origin <current-branch>
   ```

4. **PR作成**
   - 概要、変更内容、テスト、影響範囲を記載
   - GitHub CLIで作成

5. **レビュー依頼**
   - /pr-review でセルフレビューを実施
